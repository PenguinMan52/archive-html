What is HTML?

Html is a hypertext markup language. To oversimplify, it can be considered the backbone that all other web development languages sit on. The main thing you need to know is that it follows a specific structure that must be adhered to.

Here are a few of the things you'll learn in this course:

  • Document Structure
  • Layout Tags
  • Text Tags
  • Lists
  • Forms
  • Tables
  • Media
  • Links
  • Misc. Secrets

Sounds like a lot right? We're gonna make this simple by starting from structure down. To begin with, I'm going to explain what exactly we're doing here and how to get started.
